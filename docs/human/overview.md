# FinHub — обзор

FinHub помогает вести личные финансы: записывать доходы и расходы, следить за бюджетами, ставить цели и смотреть отчёты. Основной интерфейс для ежедневного использования — **Telegram-бот**.

## Что уже работает

### Учёт операций
- Доходы и расходы с категориями
- Быстрый ввод текстом: `500 кофe`, `+10000 зарплата`
- **Голосовой ввод** (opt-in): «запиши расход 300 рублей на продукты» — нужны `VOICE_ENABLED=True` и `OPENAI_API_KEY`
- Редактирование суммы, даты, комментария через inline-кнопки

### Бюджеты
- Месячные лимиты по категориям
- План/факт, остаток, процент выполнения
- Управление из главного меню бота и настроек категорий

### Цели (конверты)
- Создание целей, пополнение и снятие
- Расчёт «свободных» средств за месяц

### Отчёты
- Месячные отчёты в боте
- Экспорт в Excel (несколько листов + базовые графики)

### API и админка
- REST API (категории, транзакции, бюджеты)
- Django Admin для управления данными и текстами бота

## Что планируется

- Расширение голоса: бюджеты, цели, финансовый консультант
- Финансовый консультант на базе LLM
- Веб-дашборд с графиками

## Архитектура (кратко)

```
Telegram ←→ Bot (python-telegram-bot)
                ↓
           Django + PostgreSQL
                ↓
           REST API (для будущего фронтенда)
```

Локально и на сервере всё поднимается через **Docker Compose**: web, bot, PostgreSQL, Redis.

## Стек

Python 3.11 · Django 4.2 · PostgreSQL · Redis · Poetry · Docker
